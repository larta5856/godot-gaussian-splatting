## 📦 What is godot-gaussian-splatting?

godot-gaussian-splatting is a plugin for the Godot Engine that adds real-time 3D Gaussian Splatting rendering. This technology allows you to create smooth and detailed 3D visuals by using Gaussian functions to render scenes efficiently. It works in real time, meaning you can see changes as you make them without delay.

The plugin supports Windows computers and integrates directly with the Godot Engine, a popular tool used to build games and interactive 3D projects. Whether you want to improve your projects or explore new rendering techniques, this plugin gives you the tools to add high-quality 3D effects quickly.

## 3. Install the Plugin

- In your Godot project, click the **Project** menu at the top.
- Select **Project Settings**.
- Go to the **Plugins** tab.
- Click **Install Plugin** or **Add Existing Plugin** depending on your Godot version.
- Navigate to the downloaded plugin file and select it.
- Once added, toggle the plugin’s status to **Active**.

The plugin is now installed in your Godot project.
